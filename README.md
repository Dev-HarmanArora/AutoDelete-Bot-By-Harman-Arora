==========================================
HOW TO USE
==========================================

1. Open config.txt

2. Change USER_ID
   Example:
   USER_ID=235088799074484224

3. Change CHANNEL_ID
   Example:
   CHANNEL_ID=1498402882737934648

4. Change DELETE_AFTER

Examples:
60000 = 1 Minute
300000 = 5 Minutes
600000 = 10 Minutes

5. Save the file.

6. Restart the bot.

Done!

The bot will now automatically delete messages from the selected User/Bot
only inside the selected channel.
